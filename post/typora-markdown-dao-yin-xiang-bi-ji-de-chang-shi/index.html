Typora Markdown到印象笔记的尝试

https://leonkeen.github.io/post/typora-markdown-dao-yin-xiang-bi-ji-de-chang-shi
https://leonkeen.github.io

  热度: loading...

因为最近听课都是用的Typora先本地做好笔记，继而直接复制粘贴到印象笔记里，这样子排版好看，同时条理比较清楚。

不过有个小问题，每次Markdown里的图片没法直接导到印象笔记。

所以采取一个折中的方案：

Hexo搭建本地博客，用印象剪藏到印象笔记里。

一、安装配套环境

  1. 下载nodejs环境 http://nodejs.cn/download/

  2. 一路默认安装，同时记得需要进系统高级系统设置，打开弹出窗口的环境变量，找到path进行编辑（path添加nodejs的安装目录），另外需要重启一下电脑。

  3. 用cmd命令输入一下node看安装完成否

  4. 安装cnpm

    npm install -g cnpm --registry=https://registry.npm.taobao.org

二、安装Hexo

  1. 新建本地博客位置（自选，最好全英文目录），cd切到这个目录，输入指令

    npm install -g hexo-cli

  2. 初始化hexo

    hexo init hexo

  3. 安装依赖文件+编译资源，部署形成静态文件

    cd hexo

    cnpm install

    hexo generate

三、本地测试

开启本地博客的指令：

hexo serve

在浏览器输入：http://localhost:4000/ 即可访问到我们搭建好的hexo站点。

如果端口被占用 可以使用如下命令启动：

hexo s -p 5000

四、复制文章发布测试

文章目录：hexo\source_posts

另外，Markdown文件的表头需要写时间头

title: post1
date: 2020-08-04 21:57:21
tags:

文章插图目录：hexo\public\2020\08\04（时间戳可能一直在变化，可以多做测试）

最后开起来，可以剪藏不过有闪退现象，印象笔记还是收到了剪藏的文章。

还是比较适用于图片特别多的文章。
