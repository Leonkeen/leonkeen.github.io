搭建SSR总结教程

第一步 购买服务器

直接在vultr购买服务器. 注意要先充值和填写账单地址,当然随便填也没什么所谓.
另外我直接在谷歌搜来了一个拉新免费的credit $50,选择deploy服务器,并不能生效,所以还是需要先充值的,可以直接支付宝或者微信支付10刀即可.

选择$5的服务器 Cloud Compute
location选择美国任意位置
CentOs 选择centos 6 x64版本

直接deploy

第二步 部署VPS服务器

A. Xshell直连主机

安装Xshell
新建会话,直连到刚刚买的服务器IP

如果xshell连不上服务器，没有弹出让你输入用户名和密码的输入框，表明你开到的ip是一个被封的ip，遇到这种情况，重新开新的服务器，直到能用xshell连上为止，耐心点哦！如果同一个地区开了多台服务器还是不行的话，可以换其它地区。

由于第一次使用,还有遇到denied的情况,

The remote SSH server rejected X11 forwarding request.
解决方法就是 点击”属性“中的”隧道“，然后将下面”转发x11到连接(x)“的选项去掉即可.

B. SSR脚本端口设置

yum -y install wget

wget --no-check-certificate https://freed.ga/github/shadowsocksR.sh; bash shadowsocksR.sh

端口可以默认,也可以设置.

关于端口的设置，总的网络总端口有6万多个，理论上可以任意设置。但是有的地区需要设置特殊的端口才有效，一些特殊的端口比如80、143、443、1433、3306、3389、8080。

至于协议之类的,可以直接默认.

之后想修改账号信息，直接输入快捷管理命令：bash ssr.sh 进入管理界面，选择相应的数字来进行一键修改。

C. 一键加速服务器

先更换内核脚本

wget -N --no-check-certificate https://freed.ga/kernel/ruisu.sh && bash ruisu.sh

这个过程有点久.
锐速安装脚本


wget -N --no-check-certificate https://github.com/91yun/serverspeeder/raw/master/serverspeeder.sh && bash serverspeeder.sh

第三步 SSR连接上网

下载GitHub上的小飞机

直接设置即可!
